# Contributing to Noto Emoji Animated GIFs Collection

Thank you for your interest in contributing! We welcome contributions from the community.

## How to Contribute

### Reporting Issues

If you encounter any problems or have suggestions:

1. Check if the issue already exists in the [Issues](https://github.com/DeLTa-X-Tunisia/noto-emoji-animation-GIF/issues) section
2. If not, create a new issue with:
   - A clear, descriptive title
   - Detailed description of the problem or suggestion
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Your environment (OS, Python version, etc.)

### Suggesting Enhancements

We're always looking for ways to improve! Consider:

- New features for the download script
- Better organization or naming conventions
- Additional documentation or examples
- Performance improvements
- New use cases or integrations

### Pull Requests

1. **Fork the repository** and create a new branch from `main`
2. **Make your changes** with clear, concise commits
3. **Test your changes** thoroughly
4. **Update documentation** if needed
5. **Submit a pull request** with:
   - Description of changes
   - Why the changes are needed
   - Any related issues

### Code Style

- Follow PEP 8 for Python code
- Use clear, descriptive variable names
- Add comments for complex logic
- Keep functions focused and small

### Testing

Before submitting:

- Test the download script with different options
- Verify the script works with Python 3.6+
- Check that documentation is accurate

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/noto-emoji-animation-GIF.git
cd noto-emoji-animation-GIF

# Create a branch
git checkout -b feature/your-feature-name

# Make changes and test
python3 download_emojis.py --help

# Commit and push
git add .
git commit -m "Description of your changes"
git push origin feature/your-feature-name
```

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on what's best for the community
- Show empathy towards other community members

## Questions?

Feel free to open an issue for any questions about contributing!

## Recognition

All contributors will be recognized in the project. Thank you for making this project better!
